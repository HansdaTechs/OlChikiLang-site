Introduction

OlChikiLang is a programming language designed for the Santali language using the Ol Chiki script. Its goal is to make programming more accessible to Santali speakers by allowing them to write code using their native writing system..

ᱚᱞ ᱪᱤᱠᱤ Script Reference

OlChikiLang uses native Ol Chiki characters and Santali numerals. Developers can write programs using their own writing system.

About OlChikiLang

OlChikiLang is a programming language built using the Ol Chiki script for Santali speakers. It aims to make programming more accessible by allowing developers to write code in their native script.

The project focuses on native language computing, education, and creating new possibilities for technology through cultural identity.
